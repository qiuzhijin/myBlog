Python安装

1 双击python-3.8.6-amd64.exe安装。

2 界面，选择自定义安装，两个都打勾。

3 选择全部功能

4 下一步选择前6项，然后点击install安装。

5 安装完成，点击colse。

6 测试是否安装成功，打开cmd窗口，输入python -V。（V大写），可以看到Python 3.8.6表示安装成功。继续下一步操作。

7 在cmd窗口输入python -m pip install --upgrade pip 回车运行

8 安装结束后，继续输入：pip install jupyter

9 安装完成后输入：jupyter notebook 回车

10 如果打开浏览器进入一下界面，表示安装成功。

(End)
